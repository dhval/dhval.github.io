Jekyll post.excerpt layout

I was able to build this blog using Jekyll in less than an hour. Run into setting up posts and found quick answer on Stack Overflow.

The easiest way to setup you your post headers is to add below yml at the top of your page. This is will override everything else.

---
layout: post
title: Jekyll post.excerpt layout
date: 4 March, 2010
excerpt: I was able to build this blog using Jekyll ...
---
